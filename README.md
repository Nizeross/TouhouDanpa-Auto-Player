# TouhouDanpa-Auto-Player
一款安卓单机策略游戏东方弹破的自动挂机脚本，基于Python+ADB

这是一个基于 Python + ADB + OpenCV 的安卓游戏自动化脚本。目标是实现自动躲避弹幕、自动攻击并打破生存记录。

## 🛠️ 当前功能
- [o] 自动连接安卓模拟器 (ADB)
- [o] 识别 7x7 网格地图状态
- [o] 识别玩家与敌人位置
- [ ] 智能寻路算法 (目前只有随机移动)
- [ ] 自动释放技能 (等待开发)

## 💻 运行环境
1. **分辨率要求：** 模拟器必须设置为 **1080x1920 (DPI 480)**。
